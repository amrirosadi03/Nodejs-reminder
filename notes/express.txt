1.expressJS -> fast, unopionated, minimalist web framework for nodeJS
2. web app framework yang dibuat di atas nodeJS
3. menyediakan antarmuka yang minimal untuk kakas yang diperlukan dalam membuat sebuah aplikasi web.
4. membantu pengelolaan aliran data dari serber ke aplikasi
5. MERN, MEAN, MEVN Stack (MongoDB, expressJS, React | Angular | Vue, nodeJS)
6. Diciptakan oleh TJ Holowaychuck, pada tanggal 22 Mei 2010

Fitur expressJS
1. Menangani request dengan berbagai metode HTTP dengan mudah (Routes)
2. MVC (Model-View-Controller)
3. Terintegrasi dengan "view" rendering engine untuk mengelola template
4. Middleware


unopinionated

2. menentukan sendiri struktur aplikasi
